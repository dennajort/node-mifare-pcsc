node-mifare-pcsc
================

Use mifare 1K card with node-pcsclite
* fix
	* Can use windows8.1. (Use ACR122U)
	![alt text][pcsc_error]
	![alt text][pcsc_ok]

## Dependencies

* [node-pcsclite 0.4.8] (https://github.com/santigimeno/node-pcsclite) .Starting with version > 0.4.0 it works in Linux, OS X and Windows.
